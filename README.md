# PointGLiPE

This is the GitHub repository for the PointGLiPE project.

> ⚠️ **Note**  
> Currently, this repository only contains code and training logs for the following datasets:
> - `ModelNet40`
> - `ScanNetV2`

The full codebase, training scripts, and environment configuration will be released after the paper is accepted.

## Data Description
- `ModelNet40`: Contains the ModelNet40 dataset used in experiments along with related training logs.
- `ScanNetV2`: Contains the ScanNet v2 dataset used in experiments along with related training logs.

## Paper Information
- Title: **Rethinking and Improving Position Encoding for Point Cloud: A Global-Local Integrated Framework**  
- Submitted to: Knowledge-Based Systems (KNOSYS)
